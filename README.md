# HousiSense: Bridging Urban Context and Human Perception with a Spatial-Cognitive RAG Agent

Official repository for the paper: "HousiSense: A Spatial-Cognitive RAG Agent for Explainable Housing Reasoning Bridging Urban Context and Human Perception".

HousiSense is a novel, context-aware reasoning framework that integrates heterogeneous urban geospatial data with user-generated reviews to decode the qualitative dimensions of residential satisfaction.

Key Features
* Spatial-Cognitive RAG Architecture: Grounds Large Language Models (LLMs) in empirical spatial data to minimize hallucinations in geospatial reasoning.

* Urban Paradox Navigation: Mathematically optimizes trade-offs between extreme accessibility (transit/amenities) and residential comfort (noise/privacy).

* Multimodal Interaction: Seamlessly translates natural language queries into explainable textual reasoning and dynamic map visualizations.

* H3-based Data Fusion: Utilizes Uber’s H3 hexagonal spatial indexing for fine-grained urban service profiling.
